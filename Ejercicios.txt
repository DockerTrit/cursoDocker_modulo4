Este ejercicio es libre. Puedes tanto utilizar Dockerfile como docker-compose o ambos.

- Crear un servidor apache corriendo por el puerto 81 en el host. 81:80. 
  Crear un servidor nginx (proxy) corriendo por el puerto 80:80 de forma que vamos a enrutar el trafico entre el proxy y el servidor web.
  Configurar el fichero de nginx que haga de reverse proxy.
  Cuando se acceda a dicho servicio via navegador web tanto por dominio como direccion IP por el puerto 80 mostrara el contenido del servidor web.
